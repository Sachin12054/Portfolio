# Portfolio

Personal portfolio site built with Vite, React, TypeScript, and Tailwind CSS.

## Getting Started

1. Install dependencies:

```sh
npm install
```

2. Start the development server:

```sh
npm run dev
```

3. Build for production:

```sh
npm run build
```

4. Preview the production build:

```sh
npm run preview
```

## Tech Stack

- Vite
- React
- TypeScript
- Tailwind CSS
- shadcn/ui
